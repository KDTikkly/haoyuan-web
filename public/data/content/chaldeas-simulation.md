# 迦勒底亚斯：拟态地球演算中枢

## 项目概述

迦勒底亚斯模拟系统是一套基于 WebGL2 双轨渲染管线构建的实时行星环境演算框架，将程序化地形生成与超分辨率后处理技术融合为一体，探索实时渲染在物理极限与美学边界上的双重突破。

## 核心架构：双轨渲染管线

### Rail-A — 程序化地形推演引擎
- **多八度 FBM 噪声函数**：通过叠加 6 层不同频率与振幅的分形布朗运动噪声，实时生成大陆板块的宏观轮廓与微观地貌细节
- **动态板块漂移**：噪声种子随时间轴连续演化，驱动大陆形态在渲染帧内发生可感知的拟态漂移
- **双层混合采样**：陆地层（高振幅低频）与海洋层（低振幅高频）独立噪声场叠加，构建真实的海陆分界梯度

### Rail-B — CAS 超分辨率后处理级
- **CAS（Contrast Adaptive Sharpening）算法**：自适应检测低采样像素基底中的边缘对比度，在不引入振铃效应的前提下还原高频细节
- **突破物理分辨率极限**：以 0.1×–1.0× 可变基础分辨率渲染，经 CAS 后处理还原为工业级清晰度输出
- **四档精确控制台**：
  - `0.1×` NearestFilter — 马赛克废墟美学，像素方块感最大化
  - `0.35×` — 低分辨率轮廓可见，细节模糊
  - `0.7×` — 过渡态，CAS 增益显著
  - `1.0×` — 高频 FBM 细节全量注入，极致清晰

## 设计哲学

> 从「数字废墟」到「拟态地球」的阶梯式视觉跨越，完整诠释实时渲染的物理与美学边界。

该系统的核心命题不是「如何渲染得更好」，而是**让降质过程本身成为可观测的视觉叙事**——用户通过四档分辨率滑轨，亲历一颗星球从像素噪声中自发涌现的全过程，体验计算美学的原始张力。

## 技术栈

| 模块 | 技术 |
|------|------|
| 渲染管线 | WebGL2 / GLSL ES 3.0 |
| 地形生成 | 多八度 FBM 程序化噪声 |
| 超分辨率 | CAS（Contrast Adaptive Sharpening）|
| 光线步进 | Ray Marching 球体 SDF |
| 前端框架 | Vue 3 + TypeScript |

## 视觉参考

系统在最高分辨率档位下，可清晰呈现：
- 大陆与海洋的软渐变边界
- 高原地带的噪声纹理层叠
- 极地冰盖的高频细节涌现
- 行星轨道环的像素级锐利边缘
