# "Fate Only" — Large-Scale IP Offline Event

## Project Overview

**Fate Only** is a comprehensive fan convention for the Fate IP community, encompassing exhibitions, figure trading, fan-creation showcases, and themed performances. This second edition scaled 3× vs. the inaugural event, ultimately drawing **1,200+ attendees** — exceeding projections by 40%.

## Lessons Learned

The key to a successful IP offline event is not scale — it's the **density of community belonging**. Immersive scene design (e.g., dedicated photo installations) delivers higher word-of-mouth ROI than paid advertising spend.
