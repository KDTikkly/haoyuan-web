# 股票投资组合 LSTM AI 预测

## 项目目标

构建一套基于 **LSTM（长短期记忆网络）** 的多资产投资组合预测系统，目标不是精准预测单只股票涨跌，而是在组合层面优化夏普比率与最大回撤控制。

## 方法论

### 特征工程
- 技术面：MA5/10/20、MACD、RSI、布林带宽度
- 基本面：PE-TTM、ROE、营收同比增速
- 情绪面：北向资金净流入、融资余额变化

### 模型架构
```
Input (60 days × N features)
       ↓
LSTM Layer 1 (256 units, return_sequences=True)
       ↓
Dropout (0.3)
       ↓
LSTM Layer 2 (128 units)
       ↓
Dense (64) → ReLU
       ↓
Output: 次日收益率预测
```

### 回测框架
- 滚动训练窗口：252 个交易日
- 持仓策略：多因子打分 → Top-K 等权组合
- 交易成本：双边 0.15%（含印花税）

## 回测结果

| 指标 | 策略 | 沪深300基准 |
|------|------|-------------|
| 年化收益 | 23.4% | 8.1% |
| 夏普比率 | 1.42 | 0.61 |
| 最大回撤 | -18.3% | -31.5% |
| Calmar | 1.28 | 0.26 |

> ⚠️ 以上为历史回测数据，不构成投资建议。
